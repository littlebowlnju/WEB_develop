# WEB_develop
the repository for NJU-18SE web front-end development course homework

## homework1 - html+css
一共实现四个页面，包括注册、登录页面、主页面和二级页面
其中主页面即导航栏中的首页，首页是关于网站的简介
二级页面即导航栏中的资源，也可从首页中“前往资源区”进行跳转
登录页面可从导航栏右上角点击跳转
注册页面可从登录页面中“注册账号”点击跳转
从登录、注册页面左上角的网站logo可以跳转至首页
